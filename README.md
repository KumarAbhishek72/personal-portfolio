# Personal Portfolio

This is a simple HTML and Tailwind CSS based portfolio website.

## Structure
- `src/index.html`: Main HTML file
- `src/global.css`: Global styles using Tailwind
- `assets/pics/`: Images and assets

## Getting Started
1. Clone the repository
2. Open `src/index.html` in your browser
3. Edit content and styles as needed

## Features
- Responsive design with Tailwind CSS
- Easy to customize
